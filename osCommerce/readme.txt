Updated Inovio OS Commerce Plugin version 2.3.4.1

Instructions:

Copy the two included files to the appropriate places under your OS Commerce
install directory.  In the OS Commerce admin, find the option under Modules->Payment
for "Inovio payment Gateway."


========================Setup merchant information=================


After install Inovio payment module click on Edit button to fill merchant related information.

========================Full refund features=======================

For full refund functionality replace orders.php from admin folder.
